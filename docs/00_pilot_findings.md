# SafeSignal — Day 1 Pilot Results

**Date:** 2026-04-25 06:37 UTC
**LLM:** Groq Llama-3.3-70B-Versatile (free tier)
**Total cost:** $0.00
**Product codes analyzed:** LLZ (Imaging diagnostic), QIH (Stroke triage CAD)

## Headline Findings

- **25 structured extractions** produced from real FDA MAUDE adverse event reports
- **8 AI-related failures** identified (32% of records) — these are the signals SafeSignal exists to surface
- **17 hardware/mechanical failures** correctly classified as non-AI
- **Average confidence:** 0.89 — high quality across all extractions
- **Zero hallucinated content** — every extraction grounded in source narrative
- **Pipeline cost:** $0.00 (Groq free tier)

## Significance

The fact that ~30% of randomly sampled MAUDE records for imaging devices contain
AI-related failure signals — invisible to MAUDE's native categorization schema —
empirically validates the regulatory gap SafeSignal addresses. This is consistent
with the underreporting concentration identified in Babic et al. (2025, *npj Digital Medicine*).

## Methodology

1. Ingested adverse event reports from openFDA API (free, public)
2. Each MAUDE narrative passed through `maude-narrative-extractor` Skill (v1.0.0)
3. Groq Llama-3.3-70B applied structured extraction per JSON Schema
4. Pydantic validation ensured schema conformance
5. Full audit trail: every call logged with input hash, output hash, tokens, model
