## Significance

50% of randomly sampled MAUDE records for imaging and stroke-triage AI devices
contain AI-related failure signals — invisible to MAUDE's native categorization
schema. This empirically validates the regulatory gap MaudeSignal addresses.

The discrimination is balanced (11 AI / 11 non-AI), suggesting the extraction
Skill correctly distinguishes software/algorithm failures from hardware/mechanical
failures without bias toward either category.

## Methodology

1. Adverse event reports ingested from openFDA API (free, public)
2. Each MAUDE narrative passed through `maude-narrative-extractor` Skill v1.0.0
3. Groq Llama-3.3-70B applied structured extraction per JSON Schema
4. Pydantic validation ensured schema conformance
5. Full audit trail: every call logged with input hash, output hash, tokens, model
