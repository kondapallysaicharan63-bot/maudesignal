## The One-Minute Summary

Document 1 answers **WHY.**
Document 2 defines **WHAT and HOW MUCH.**

Document 4 lists **WHAT YOU NEED TO LEARN.**

## The Three Non-Negotiable Rules

1. **No code before docs.** All 8 documents must be drafted and committed before Week 2 (first production code).
2. **No feature #8.** The Charter §4 has 7 features. Adding one requires removing one.
3. **No skipping discovery.** Gate 0 at end of Week 1 is a binary pass/fail on customer discovery. No exceptions.
